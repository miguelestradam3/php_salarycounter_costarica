# 💰 Salary Counter Costa Rica

A simple web application that helps users estimate and compare salaries in Costa Rica. The calculator provides an easy-to-use interface for performing salary-related calculations directly in the browser.

## 🚀 Features

- Calculate salary values
- Simple and responsive interface
- Fast client-side calculations
- Designed for Costa Rican salary estimates

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/miguelestradam3/salarycounter_costarica.git
```

Open `index.html` in your preferred web browser.
